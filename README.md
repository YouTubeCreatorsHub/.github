![image](https://github.com/user-attachments/assets/7ac3fcd9-595b-420d-9720-310d678370f2)# 🎨 YouTube Creators Hub

크리에이터를 위한 통합 플랫폼

## 🌟 주요 기능

- 📊 채널 분석 대시보드
- 👥 크리에이터 커뮤니티
- 📈 콘텐츠 성과 분석
- 🤝 멘토링 시스템
- 🔔 실시간 알림
- 📱 모바일 웹 지원

## 🛠️ 기술 스택

### Backend
- Spring Boot
- PostgreSQL
- JPA
- Spring Security

### Frontend
- React
- TypeScript
- Tailwind CSS
- Recoil/Redux

### DevOps
- AWS
- GitHub Actions
- Docker
